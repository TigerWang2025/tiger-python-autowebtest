# tiger-python-autowebtest

第一步：创建项目
第二步：安装selenium
1、在Pycharm的命令控制台执行：pip install selenium，等待安装完成
2、检查是否安装：pip show selenium，待显示版本信息即可
第三步：安装pytest
1、在Pycharm的命令控制台执行：pip install pytest，等待安装完成
2、检查是否安装：pip show pytest，待显示版本信息即可



其它插件的安装：

pip install xToolkit

pip install pandas
